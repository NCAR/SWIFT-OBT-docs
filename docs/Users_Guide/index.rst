**Foreword: A note to SWIFT-OBT users**

This User's Guide is provided as an aid to users of the SAMURAI Winds and Intensity in Forecasting Time Objective Best Track (SWIFT-OBT) system. SWIFT-OBT is an integrated observational analysis which provides forecaster support products for the National Hurricane Center. More details about the SWIFT-OBT project can be found on the
`SWIFT-OBT Project website <http://github.io/NCAR/tcobs>`_.

It is important to note here that SWIFT-OBT is still under active development.
This documentation describes the |release| release dated |release_date|.
Intermediate releases may include bug fixes.
